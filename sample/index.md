## MdPad 기능 소개

MdPad는 [[getting-started|시작 가이드]]에 설명된 것처럼 로컬 마크다운 파일을 위한 뷰어/편집기입니다.

### 핵심 기능

| 기능 | 단축키 |
|------|--------|
| Quick Switcher | Ctrl+P |
| 전문 검색 | Ctrl+Shift+F |
| 저장 | Ctrl+S |
| 보기 모드 | V |
| 분할 모드 | S |
| 편집 모드 | E |

### 태그

이 노트는 #샘플 #소개 #mdpad 태그를 포함합니다.
