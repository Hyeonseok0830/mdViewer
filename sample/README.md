# mdViewer

로컬 마크다운 파일을 실시간으로 렌더링하는 뷰어/편집기입니다.

## 기능

- **실시간 미리보기** — 파일을 저장하는 순간 자동 갱신
- **구문 강조** — 80여 가지 프로그래밍 언어 지원 (shiki)
- **수식 렌더링** — KaTeX로 LaTeX 수식 표현
- **다이어그램** — Mermaid 차트 지원
- **편집** — 내장 에디터로 바로 수정 후 저장

## 단축키

| 키 | 동작 |
|---|---|
| `V` | 미리보기 모드 |
| `S` | 분할 편집 모드 |
| `E` | 전체 편집 모드 |
| `Ctrl+S` | 저장 |
| `Ctrl+O` | 파일 열기 |
| `Ctrl+Shift+O` | 폴더 열기 |

## 시작하기

왼쪽 사이드바에서 파일을 클릭하거나, 헤더의 `✏` 버튼으로 편집 모드로 전환하세요.
