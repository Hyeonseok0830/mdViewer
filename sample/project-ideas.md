## 아이디어 목록

### 1. 개인 지식베이스

마크다운 파일로 개인 위키를 만들어보세요.
[[markdown-guide]]의 Wiki Links 기능을 활용하면 노트 간 연결이 쉽습니다.

**장점:**
- 로컬 파일이라 프라이버시 보장
- Git으로 버전 관리 가능
- [[getting-started|MdPad]]로 언제든 탐색 가능

### 2. 프로젝트 문서화

```
docs/
├── index.md          # 프로젝트 개요
├── architecture.md   # 아키텍처 설명
├── api.md           # API 문서
└── changelog.md     # 변경 이력
```

### 3. 학습 노트

강의나 책을 읽으며 노트를 작성하고, Wiki Links로 개념을 연결하세요.

$$
\text{지식} = \sum_{i=1}^{n} \text{노트}_i \times \text{연결}_i
$$
