# GitHub Flavored Markdown

## 텍스트 서식

**굵게**, *기울임*, ~~취소선~~, `인라인 코드`, [링크](https://github.com)

> 인용구는 이렇게 표현합니다.
> 여러 줄도 가능합니다.

## 목록

### 순서 없는 목록
- 항목 1
  - 중첩 항목
    - 더 깊은 중첩
- 항목 2
- 항목 3

### 순서 있는 목록
1. 첫 번째
2. 두 번째
3. 세 번째

### 체크리스트
- [x] 실시간 미리보기 구현
- [x] 구문 강조 (shiki)
- [x] 수식 렌더링 (KaTeX)
- [x] 다이어그램 (Mermaid)
- [x] 파일 편집 기능
- [ ] Microsoft Store 배포

## 표

| 언어 | 패러다임 | 특징 |
|------|----------|------|
| TypeScript | 정적 타입 | 타입 안전성 |

| Go | 동시성 | 단순함 |
