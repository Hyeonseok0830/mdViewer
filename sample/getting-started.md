# 시작 가이드

> [!TIP]
> MdPad를 처음 사용하신다면 이 문서부터 읽어보세요!

## 설치 및 실행

1. `MdPad-Setup-2.0.0.exe` 를 실행해 설치합니다.
2. 설치 후 `.md` 파일이나 폴더를 MdPad로 열면 바로 시작됩니다.

## 기본 사용법

MdPad는 [[index|메인 페이지]]에서 설명한 것처럼 마크다운 파일을 실시간으로 렌더링합니다.

### 폴더 열기

폴더를 열면 왼쪽에 **파일 트리**가 표시됩니다.
파일을 클릭하면 즉시 미리보기가 열립니다.

### Wiki Links

다른 노트를 `[[파일명]]` 형식으로 링크할 수 있습니다:
- [[markdown-guide]] 에서 마크다운 문법 확인
- [[project-ideas]] 에서 아이디어 탐색

### 그래프 뷰

헤더의 **◉** 버튼을 누르면 노트 간 연결 관계를 그래프로 볼 수 있습니다.
[[index]], [[markdown-guide]], [[project-ideas]] 가 어떻게 연결되는지 확인해보세요.

## 단계별 학습

1. 이 폴더를 MdPad로 열기
2. 왼쪽 트리에서 파일 탐색
3. **Ctrl+P** 로 Quick Switcher 열기
4. **◉** 버튼으로 그래프 뷰 열기
