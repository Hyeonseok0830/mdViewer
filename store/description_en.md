# mdViewer — Markdown Viewer & Editor

## Short Description (max 200 chars)

A real-time Markdown viewer and editor with syntax highlighting, math (KaTeX), and diagrams (Mermaid). Renders beautifully as you type.

---

## Full Description (max 10,000 chars)

**mdViewer** is a local Markdown viewer and editor built for developers and writers.
The preview updates the instant you save, so you can write Markdown comfortably without opening a separate browser.

### Key Features

**Real-Time Preview**
- Preview refreshes automatically on every file save
- When using the built-in editor, preview updates as you type

**Beautiful Syntax Highlighting**
- GitHub-style light/dark theme with automatic switching
- 80+ languages: JavaScript, TypeScript, Python, Rust, Go, and more
- Accurate token parsing powered by the shiki engine

**Math Rendering (KaTeX)**
- Inline math: `$E = mc^2$`
- Block math: matrices, integrals, series — full LaTeX syntax supported

**Diagrams (Mermaid)**
- Flowcharts, sequence diagrams, Gantt charts, state diagrams
- Generate professional diagrams with a single code block

**Folder Explorer**
- Opening a folder shows a file tree in the left sidebar
- One-click file switching
- Automatic table of contents extraction with click-to-scroll

**Edit Mode**
- Three modes: Preview / Split (Edit + Preview) / Edit only
- Tab indentation, Ctrl+S instant save

**Full GitHub Flavored Markdown Support**
- Tables, checklists, strikethrough, footnotes
- frontmatter YAML parsing and display

**Dark / Light Theme**
- Auto-detects system preference
- Manual toggle button in the header

### How to Use

1. Launch the app → select a file (.md) or folder
2. Click files in the left sidebar
3. Switch to edit mode with the ✏ button in the header
4. Save with Ctrl+S

### Keyboard Shortcuts

- V: Preview mode
- S: Split mode (Edit + Preview)
- E: Edit-only mode
- Ctrl+S: Save
- Ctrl+O: Open file
- Ctrl+Shift+O: Open folder
