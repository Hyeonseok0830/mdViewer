# MdPad — 마크다운 뷰어/편집기

## 간단한 설명 (짧은 설명, 최대 200자)

마크다운 파일을 아름답게 렌더링하는 실시간 뷰어/편집기. 구문 강조, 수식(KaTeX), 다이어그램(Mermaid) 지원.

---

## 전체 설명 (최대 10,000자)

**MdPad**는 개발자와 작가를 위한 로컬 마크다운 파일 뷰어/편집기입니다.
파일을 저장하는 순간 미리보기가 자동으로 갱신되어, 별도의 브라우저 없이 편리하게 마크다운을 작성할 수 있습니다.

### 주요 기능

**실시간 미리보기**
- 파일 저장 즉시 미리보기 자동 갱신
- 내장 편집기로 작성하면 타이핑 중에도 실시간 미리보기

**아름다운 구문 강조**
- GitHub 스타일 라이트/다크 테마 자동 전환
- JavaScript, TypeScript, Python, Rust, Go 등 80여 개 언어 지원
- shiki 엔진 기반 정확한 토큰 파싱

**수식 렌더링 (KaTeX)**
- 인라인 수식: `$E = mc^2$`
- 블록 수식: 행렬, 적분, 급수 등 LaTeX 전체 문법 지원

**다이어그램 (Mermaid)**
- 플로우차트, 시퀀스 다이어그램, 간트 차트, 상태 다이어그램
- 코드 블록 한 줄로 전문적인 다이어그램 생성

**폴더 탐색기**
- 폴더를 열면 왼쪽 사이드바에 파일 트리 표시
- 클릭으로 빠른 파일 전환
- 목차(TOC) 자동 추출 및 클릭 이동

**편집 모드**
- 미리보기 / 분할(편집+미리보기) / 편집 전용 세 가지 모드
- Tab 들여쓰기, Ctrl+S 즉시 저장

**GitHub Flavored Markdown 완전 지원**
- 표, 체크리스트, 취소선, 각주
- frontmatter YAML 파싱 및 표시

**다크/라이트 테마**
- 시스템 설정 자동 감지
- 수동 전환 버튼 제공

### 사용 방법

1. 앱 실행 → 파일(.md) 또는 폴더 선택
2. 왼쪽 사이드바에서 파일 클릭
3. 헤더의 ✏ 버튼으로 편집 모드 전환
4. Ctrl+S로 저장

### 단축키

- V: 미리보기 모드
- S: 분할 모드 (편집 + 미리보기)
- E: 편집 전용 모드
- Ctrl+S: 저장
- Ctrl+O: 파일 열기
- Ctrl+Shift+O: 폴더 열기
